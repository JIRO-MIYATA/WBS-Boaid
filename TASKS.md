## 0. 前提
このシステムは以下の要件を満たすこと。

- [x] フロントエンドは React で構築する
- [x] Webブラウザから利用する社内システムとする
- [x] 利用者はメールアドレスをログインIDとして使用する
- [x] 初期パスワードは `elpa1234` とする
- [x] 初回ログイン時に必ずパスワード変更を要求する
- [x] 部門目標は年間目標であり、月次で進捗確認を行う
- [x] 日々のルーチン業務は、実施チェックおよび記録で管理する

- [x] FAQ管理機能（問い合わせ対応状況の可視化）を持つ
- [x] 権限は「一般部員」「管理者」を持つが、目標・開発・FAQは全員操作可能とする
- [x] 認証と権限制御はフロントエンドだけでなくバックエンドでも実施する
